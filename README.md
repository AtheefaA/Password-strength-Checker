# Password Strength Checker (Python)

This project is a simple Python program that evaluates the strength
of a password based on length and character complexity.

## Features
- Checks password length
- Detects uppercase, lowercase, numbers, and symbols
- Displays password strength level
- Beginner-friendly Python code

## Technologies Used
- Python
- Regular Expressions (optional)

## How It Works
The program analyzes the entered password and checks
for different character types and length to determine
whether it is weak, medium, or strong.

## Example
Input: Pass@123  
Output: Strong Password
